Date Mon 09 November 2015 Tags wasatch photonics

Web service to create calibration reports for wasatch photonics spectrometers. Use a responsive web form to accept a serial number, calibration coefficients and (optional) product imagery. Create a PDF document and graphical thumbnail, store forever.

Live demo available at waspho.com. This is used by the CookBook spectrometer tracking system. This was the first system created as part of the microwave/cookbook update. Uses travis and coveralls for continuous integration testing. Deployed on a Digital Ocean Droplet
